## oringin

index.html 在根目录下。
index.html 不在 html 文件夹里。

请双击 index.html 来浏览我的网站。
请使用谷歌或火狐浏览器。

js 文件放在两个目录里，分别是 dependencies 和 js。

dependencies 文件夹存放的是他人编写的 js 库或框架，也是我这个项目所依赖的 js 文件。
js 文件夹存放的是我自己编写的 js 代码。

本网站采用 flex 布局，能够在一定的分辨率范围内获得无需横向滚动的浏览体验。
